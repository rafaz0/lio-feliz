## 8. Core API Frozen (GOV-006)

A partir de 15/07/2026, os seguintes componentes passam a ser **congelados**:

| Componente | Arquivo |
|------------|---------|
| Result | `src/core/domain/result.ts` |
| DomainError | `src/core/domain/errors.ts` |
| ValueObject | `src/core/domain/value-object.ts` |
| EntityId | `src/core/domain/entity-id.ts` |
| Entity | `src/core/domain/entity.ts` |
| AggregateRoot | `src/core/domain/aggregate-root.ts` |
| DomainEvent | `src/core/domain/domain-event.ts` |

**Regras de congelamento:**
1. Alterações comportamentais só via ER específica + PI atualizada + EWO atualizado
2. Correções de bug permitidas se não alteram API pública
3. Refatorações comportamentais requerem fluxo completo ER → PI → EWO
4. Novas classes estendendo Core Foundation **não** afetadas pelo congelamento
