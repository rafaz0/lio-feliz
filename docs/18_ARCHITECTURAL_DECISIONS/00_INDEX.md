# Objetivo

Este documento indexa todos os Architecture Decision Records (ADR) do Lio Feliz.

Os ADRs registram decisões arquiteturais, suas alternativas, o contexto que as motivou e as consequências esperadas.

Qualquer nova decisão arquitetural significativa deverá ser registrada em um novo ADR antes de ser implementada.
